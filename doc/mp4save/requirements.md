# 需求文档

## 简介

该系统用于管理抖音直播录制任务，用户在输入直播间房号后，需要先选择 mp4 文件保存目录，再创建录制任务。

## 术语表

- **System**: 该系统
- **User**: 录制任务创建者
- **保存目录**: 用户选择用于保存录制 mp4 文件的本地目录
- **录制任务**: 系统创建并执行的直播录制任务

## 需求

### 需求1: 录制前选择保存目录

**用户故事：** 作为用户，我想在创建录制任务前选择保存目录，以便决定录制文件保存位置。

#### 验收标准

1. WHEN 用户输入直播间房号并点击创建任务 THEN THE System SHALL 弹出目录选择器要求选择保存目录
2. WHEN 用户已选择保存目录 THEN THE System SHALL 允许创建录制任务并将该目录作为保存路径
3. WHEN 用户未选择保存目录 THEN THE System SHALL 不创建录制任务并提示需要选择保存目录

### 需求2: 记住上次选择目录

**用户故事：** 作为用户，我想系统记住上次选择的保存目录，以便下次快速选择。

#### 验收标准

1. WHEN 用户成功创建录制任务且已选择保存目录 THEN THE System SHALL 记住该目录作为下次默认选项
2. WHEN 用户再次创建录制任务并弹出目录选择器 THEN THE System SHALL 默认选中上次保存目录

### 需求3: 取消选择目录

**用户故事：** 作为用户，我想在取消目录选择时不创建任务，以避免误操作。

#### 验收标准

1. WHEN 用户在目录选择器中取消 THEN THE System SHALL 取消创建录制任务并提示操作已取消
2. WHEN 取消后 THEN THE System SHALL 保持录制任务列表无新增任务
